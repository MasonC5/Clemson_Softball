# Clemson_Softball

## PlayerStats.csv

Raw game-level and player-level statistics for the 2025 Clemson Softball season.

Includes offensive metrics (hits, runs, RBI, home runs), pitching metrics (innings pitched, runs allowed, strikeouts, pitches, strikes), player class, and game metadata.

Serves as the primary data source for all analysis and visualizations in this project.

## macaste_softball.ipynb

Jupyter Notebook containing all data cleaning, aggregation, and visualization code.

Uses Python, pandas, and matplotlib to:

Analyze roster composition by class

Track season development through cumulative wins

Evaluate offensive production and pitching workload/efficiency

Quantify leadership contributions and returning vs graduating production

All figures used in the presentation are generated directly from this notebook.

## macaste_softball2025(1).pptx

Final presentation summarizing insights from the analysis.

Walks through roster overview, season development, offense, pitching, leadership, and next-season outlook.

Designed to translate quantitative findings into a clear performance narrative supported by visual evidence.
